# Security — GPS v0
**Status: alpha, mock rail, unaudited. No real funds.**
Defends (tested): tampered policy/agreement refused before payment · human-approval
gate mandatory (denial test) · unresolvable recipient stops the job with a failure
receipt · hash-chained signed ledger detects tamper/reorder · integer-exact split math.
Not yet: real Lightning · idempotent retries (fail-fast v0; see ROADMAP) · encrypted
receipts · key storage hardening (keys in memory) · SatsPath resolver (interface only).
Report privately to the maintainer; no public issues for vulns.
